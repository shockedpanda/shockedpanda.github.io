Development roadmap

We do not have fixed release dates. Instead, we build tools on request for clients, then release a stripped‑down open‑source version on GitHub for anyone to use or extend.

ToolDescriptionAvailability
KYC Document Certification ValidatorChecks certification wording, dates, and certifier details.On request (client‑driven). Open‑source after first build.
STR/SAR Drafting AssistantFlags unusual patterns and drafts suspicious transaction reports.On request. Open‑source after delivery.
Compliance LibrarianChat‑based document retrieval and expiry checking.On request. Open‑source after delivery.
Ongoing Monitoring EnginePeriodic re‑screening against sanctions/PEP and adverse media.On request. Open‑source after delivery.
Corporate Documents LibrarianSpecialised assistant for corporate entity documents.On request. Open‑source after delivery.
Board/MI Report CompilerDrafts board packs from multiple compliance sources.On request. Open‑source after delivery.
Regulatory Change Impact AssessorMaps new regulations to internal controls.On request. Open‑source after delivery.

How it works: If you need a tool that isn’t already built, we will build it for you under a pilot agreement. After delivery, we remove client‑specific customisations and publish a generic open‑source version on our GitHub organisation. That way, the community benefits and you are never locked in.
